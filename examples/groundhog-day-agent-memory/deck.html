Statelessness

The 6:00 AM clock flip
Context window

One day is all the room Phil has — anything that doesn't fit never happened.

Where it breaksPhil's day always has identical length; a context window fills as the conversation grows and can overflow mid-task.

The window is a hard budget of text; long sessions must summarize or drop the oldest material to keep fitting.

Agent memory

Where it breaksPhil's memory lives inside him; agent memory lives outside — the model itself never learns or changes.

Memory writes

Phil can't relive everything at once — each day he chooses what deserves to carry forward.

Where it breaksPhil gets unlimited retries to find what matters; a memory system must decide what to keep in real time, first pass.

A write policy scores each fact for durable value; keep too much and retrieval drowns, keep too little and the agent forgets you.

Retrieval

Knowing everything is useless unless the right memory surfaces at the exact moment it's needed.

Where it breaksPhil recalls from lived experience; retrieval is a similarity lookup that can confidently fetch the wrong memory.

Each turn the system searches the store for entries resembling the current request and injects the best matches into the window.

The Rita slap montage
Memory is not understanding

Phil has every fact about Rita memorized and still gets slapped — stored facts aren't understanding.

Where it breaksIn the film the missing piece is sincerity; in an agent it is reasoning over the memories — not feelings.

Memory only supplies raw material; the model must still reason correctly over what it retrieves, and that step can fail.

Where the analogy ends

Memory poisoning

A prompt injection that normally dies with the session instead persists in memory and resurfaces days later in a different context — a sleeper exploit with no movie parallel.

These ideas are genuinely new — the movie has no shape for them.
